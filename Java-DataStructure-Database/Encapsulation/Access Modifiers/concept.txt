Access Modifiers 

- Restrict visibility to class members 
- Most access modifiers can be applied to 
       - classes 
       - Methods 
       - Variables 
       - Constructors 

-public 
   - Accessible anywhere in the application 

-protected 
   - Accessible within the package and any subclasses 
-no modifier 
   - Access only within the package 
-private 
   - Accessble only within the class 
--default 
    -No modifier also called default or package private
